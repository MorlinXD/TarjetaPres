Valores principales de la propiedad display en CSS

Valor Descripción
inline Muestra un elemento como en línea (como <span>). Las propiedades de altura y ancho no tienen efecto.
block Muestra un elemento como un bloque (como <p>). Comienza en una nueva línea y ocupa todo el ancho disponible.
flex Muestra un elemento como un contenedor flexible de nivel bloque. Ideal para diseños flexibles.
grid Muestra un elemento como un contenedor de cuadrícula de nivel bloque. Perfecto para diseños en rejilla.
inline-block Muestra un elemento como un bloque en línea. Permite establecer altura y ancho.
none Elimina completamente el elemento del diseño (no se muestra ni ocupa espacio).
initial Establece la propiedad a su valor predeterminado.
inherit Hereda el valor de la propiedad del elemento padre.
